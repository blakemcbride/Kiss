(static) someControlValueChanged()

Source:
  • Utils.js, line 1030
This function is called to indicate to the system that some control value changed by the user. Programmatic changed to not set this. It is mainly used internally as all the Kiss controls use this function. If there is a function connected to some-control-value-changing via the setSomeControlValueChangeFunction function, it will be executed the first time only that any control value is changed. Subsequent changes will not trigger the some-control-value-changing function. The some-control-value-changing function is passed the single argument true. Non-Kiss controls should call this function if their value changes.
