(static) callAll()

Source:
  • Server.js, line 183
Used to call a number of simultaneous web services and wait till they're all done before processing any of their results. This function takes a variable number of arguments. The first argument is an array of the Promises from each web service call. Each remaining argument is a function that gets the result from the positionally corresponding promise in the first argument. If any are null there is no function executed for that returned promise. Each function that gets executed gets passed the return value of the associated web service.
