KISS JavaDoc Top Level

This is the top-level documentation for the KISS project. It documents the back-end API. This API is accessible through the Java or Groovy languages.

The KSS user manual is available at https://blakemcbride.github.io/Kiss/manual/man.

The front-end API is documented at https://blakemcbride.github.io/Kiss/manual/jsdoc/.

The KISS system was written by Blake McBride
